# Requirements — Frog Evolution: Локализация + Настройки

## v1 Requirements

### i18n Infrastructure
- [ ] **I18N-01**: Установить и настроить react-i18next с поддержкой RU/EN/ES
- [ ] **I18N-02**: Создать JSON-файлы переводов (`src/i18n/ru.json`, `en.json`, `es.json`)
- [ ] **I18N-03**: Все UI-строки вынесены в переводы (хардкод убран из компонентов)
- [ ] **I18N-04**: Имена всех 24 лягушек переведены на EN и ES по смыслу

### Language Settings
- [ ] **LANG-01**: Выбранный язык сохраняется в localStorage (`frog_lang`), дефолт `ru`
- [ ] **LANG-02**: Приложение загружается с языком из localStorage при старте

### Settings Panel
- [ ] **UI-01**: Кнопка 📖 в BottomBar открывает полноэкранный `SettingsModal`
- [ ] **UI-02**: Модал содержит две вкладки: "Бестиарий" и "Настройки"
- [ ] **UI-03**: Вкладка "Настройки" — переключатель языка (RU / EN / ES)
- [ ] **UI-04**: Вкладка "Настройки" — toggle музыки (заглушка, без функции)
- [ ] **UI-05**: Вкладка "Настройки" — toggle звуковых эффектов (заглушка)
- [ ] **UI-06**: Вкладка "Настройки" — переключатель формата денег (короткий / полный)
- [ ] **UI-07**: Вкладка "Настройки" — кнопка "Баг репорт" открывает `t.me/YOUR_USERNAME`

### Bestiary
- [ ] **BEST-01**: Вкладка "Бестиарий" показывает карточки всех 24 лягушек
- [ ] **BEST-02**: Открытые лягушки (в `discoveredLevels`) — полная карточка: имя, локация, доход/сек, размер
- [ ] **BEST-03**: Неоткрытые лягушки — силуэт (CSS grayscale/blur) + "???" вместо имени
- [ ] **BEST-04**: Имя лягушки в карточке отображается на текущем языке интерфейса

### Number Format
- [ ] **FMT-01**: Настройка формата денег применяется ко всем местам где показывается gold
- [ ] **FMT-02**: Короткий формат: `1.5K`, `2.3M`, `1.1B` — как сейчас
- [ ] **FMT-03**: Полный формат: `1,500`, `2,300,000` — без сокращений

## v2 (Deferred)
- Автоопределение языка из Telegram.WebApp.initDataUnsafe.user.language_code
- Серверное хранение языковых предпочтений
- Реальная музыка и звуковые эффекты
- Поиск/фильтр в бестиарии по локации

## Out of Scope
- Backend изменения — только клиент
- Telegram auto-detect языка — добавить позже в v2
- Язык на сервере — localStorage достаточно

### Rare Crate
- [x] **RARE-01**: Редкий «золотой» бокс спавнится реже обычного (отдельная логика/интервал в MainScene)
- [x] **RARE-02**: Тап на золотой бокс открывает React-модалку со слот-машиной (прокрутка лягушек → остановка на случайной)
- [x] **RARE-03**: После закрытия модалки выигранная лягушка спавнится на поле через `gameStore`
