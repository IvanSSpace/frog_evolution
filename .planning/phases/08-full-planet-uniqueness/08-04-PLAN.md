}

`deriveModulations(seed, _archetype)` функция: создаёт mulberry32(seed) и возвращает 5 индексов через `Math.floor(rng() * N)`.

`applyVoicing(notes, rotationIdx, inversionIdx)` helper: возвращает new array после rotation (cyclic shift по rotationIdx % notes.length) и inversion (первые inversionIdx нот +12 semitones, capped на notes.length-1).

`detuneCents(bin, range = 15)` helper: возвращает cents для bin 0..3 — используй map `[-range, -range/3, range/3, range]`.

`cutoffHz(bin, range = [400, 4000])` helper: возвращает Hz по логарифмической интерполяции — `lo * Math.pow(hi/lo, bin/3)`, округлённое.
