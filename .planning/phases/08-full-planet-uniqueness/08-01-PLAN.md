- duration per bounce: 200ms easeOut→easeIn для апекса.
- Total dur ~900ms; в onComplete финального tween — destroy.

**89. `compDigitalGlitch`** — Пиксельные RGB-shift искажения.
- 3 копии-clone маленьких прямоугольников разных RGB цветов (0xff0066/0x00ff66/0x0066ff), смещены случайно ±sys.size*0.3.
- 4-6 "stutter" шагов: каждые ~80ms перерисовывают позиции (через delayedCall + alpha pulse).
- BlendMode SCREEN. Total dur ~600ms, потом alpha→0 + destroy.

**90. `compRingPulsar`** — Пульсирующее кольцо с heartbeat-ритмом (lub-DUB).
- 1 ring graphics, strokeCircle радиуса sys.size * 1.3.
- 2 пульсации: первая короткая (0.85→1.15 scale за 120ms), вторая сильная (1.0→1.4 за 200ms).
- Цвет — `this.pickColor(rng, sys)`, alpha 0.8→0 в финале.
- Total dur ~900ms.

**91. `compSwarmParticles`** — Рой 12-20 точек огибает планету по орбите.
- N = `12 + Math.floor(rng() * 9)` particles.
- Каждая на orbital path радиусом `sys.size * (1.0 + rng() * 0.4)`, угловая скорость случайная (±0.003 рад/мс).
- Используй `this.events.on('update', update)` + dt; через 1000ms все fade out + destroy + отпиши update.

**92. `compPrismRefract`** — Спектр радуги преломляется через "грань" планеты.
- 7 разноцветных лучей расходятся из точки на краю планеты (RAINBOW: 0xff0000, 0xff7f00, 0xffff00, 0x00ff00, 0x0000ff, 0x4b0082, 0x9400d3).
- Каждый луч — line из центральной точки длиной `sys.size * (1.5 + rng() * 0.5)` под небольшим углом (±15° от base).
- Lines fade-in за 100ms, fade-out за 400ms. Total ~600ms. Под archetype crystal_bio, mineral, plasma.

**93. `compLifeBloom`** — Растущие кривые-лозы (для organic).
- 4-6 "vine" линий, каждая — Bezier-style серия lineSegments из центра наружу через `sys.size * 0.85`.
- Каждый сегмент рисуется postfix через delayedCall с шагом 60ms.
- Цвет — pickColor (organic palette даст зелёные оттенки). Total dur ~1000ms.
- В конце на каждой "ветке" — мелкая точка-цветок (fillCircle radius DPR*1.5).

**94. `compWindRibbons`** — Ленты ветра (для mist/aerial).
- 2-3 ленты-graphics, каждая — sin-wave path из 12 segments длиной `sys.size * 2.5`, движется по экрану от -size до +size.
- Используй `this.events.on('update', update)` для прогрессии вдоль траектории.
- Лента fade alpha 0→0.6→0 за свой жизненный цикл. Total dur ~1100ms.

**95. `compWreckageOrbit`** — Мелкие куски обломков на орбите (для destroyed).
- 6-10 крошечных треугольников/прямоугольников (`fillTriangle`), радиус `sys.size * (1.1 + rng() * 0.3)`.
- Все вращаются вокруг планеты с разной скоростью; через 900ms все одновременно fade и destroy.
- Используй `this.events.on('update', update)` + accumulated angles.
