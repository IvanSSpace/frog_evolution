## Pattern (как в Phase 9)

Каждый comp-метод → отдельный файл:

```typescript
// src/game/effects/anim/shared/compMultiRing.ts
import type Phaser from 'phaser'
import { DPR, pickColor, pickEase } from './sharedHelpers'
import type { AnimSys } from './types'

export function compMultiRing(
  scene: Phaser.Scene,
  sprite: Phaser.GameObjects.Container,
  sys: AnimSys,
  rng: () => number,
): void {
  // тело метода без `this.` префиксов
  // `this.tweens.add(...)` → `scene.tweens.add(...)`
  // `this.add.graphics()` → `scene.add.graphics()`
  // `this.time.delayedCall(...)` → `scene.time.delayedCall(...)`
  // `this.pickColor(rng, sys)` → `pickColor(rng, sys)`
  // `this.pickEase(rng)` → `pickEase(rng)`
}
```

В `index.ts` добавляется `export { compMultiRing } from './compMultiRing'`.

В `StarMapScene.ts` удаляется метод-тело + `import` нового comp.
В местах вызова: `this.compMultiRing(sprite, sys, rng)` → `compMultiRing(this, sprite, sys, rng)`.

## Risk notes

- Главные ловушки:
