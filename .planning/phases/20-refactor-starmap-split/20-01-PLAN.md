## Tasks

1. Создать `src/game/scenes/starmap/types.ts`, перенести туда все type/interface определения
2. Создать `src/game/scenes/starmap/helpers.ts`, перенести pure-функции и константы
3. В `StarMapScene.ts` заменить локальные определения на `import { ... } from './starmap/types'` и `'./starmap/helpers'`
4. Если `worldToDom` был методом — переписать на функцию `worldToDom(this, ...)` в местах использования
5. `npx tsc --noEmit` — must pass
6. `npm run lint` — must pass
7. Ручная проверка StarMap (открыть, кликнуть на планету, popover должен появиться)
8. Commit: `refactor(starmap): extract types and helpers (20-01)`
