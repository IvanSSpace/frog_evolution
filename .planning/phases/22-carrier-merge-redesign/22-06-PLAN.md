<objective>
Cosmos gate: вся серум-машинерия скрыта/inactive до первого L18+L18 sentinel.

Pre-cosmos gameplay = чистый merge + tractor + locations. После unlock — серум-машинерия
постепенно открывается (mega-box начинает дропать серум, SerumBar показывается, Cosmic Hub
доступен, Star Map доступна).

Purpose: новый игрок не видит overwhelming UI сразу. Прогрессия gated за достижение
(L18+L18 это требует 18 levels merge'ев). Это existing pattern из Phase 16 (progressive disclosure UX-09).

Output: hasCosmosUnlocked flag, useCosmosUnlocked hook, conditional rendering во всех
серум-связанных компонентах, hook в MergeController на L18+L18 path.

ВАЖНО: existing игроки с прогрессом — если у них уже есть discovered=19 в L25 slice,
hasCosmosUnlocked должен быть true at load. Plan 22-07 (migration) обработает legacy.
Здесь же — для NEW state.
</objective>
