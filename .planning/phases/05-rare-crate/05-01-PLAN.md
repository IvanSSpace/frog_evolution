# Plan 05-01: Rare Crate — Phaser spawn + eventBus

## Objective

Add a golden rare crate that spawns on its own timer in MainScene. When tapped, it disappears with an explosion and emits `rareCrate:opened` event (no frog yet — modal handles it). Add `rareCrate:claim` event listener so MainScene can spawn the frog after modal closes.

**4. Reset `rareBoxProgressMs` in all places where `boxProgressMs` is reset to 0** (check grep results — there are 3 locations: create(), location transition, handleLocationReset or similar).

}
```

**6. Modify `spawnBox()` signature** to accept optional `isRare` param:

private spawnBox(isRare = false) {
  // ... existing code to create img ...
  if (isRare) {
    img.setTint(RARE_BOX_TINT)
    img.setDisplaySize(BOX_DISPLAY_SIZE * RARE_BOX_SCALE_MULT, BOX_DISPLAY_SIZE * RARE_BOX_SCALE_MULT)
  }
  const box: BoxData = { img, isLanding: true, baseScale, baseY: targetY, idleTween: null, isRare }
  // ... rest unchanged ...
}
```

**7. In `onBoxTapped()`, after removing the box and starting the explosion animation, add rare crate branch:**
```typescript
if (box.isRare) {
  // Rare box: emit event, don't spawn frog here
  const state = useGameStore.getState()
  const loc = getLocationById(state.currentLocation)
  const baseLevel = getCrateLevel(state.upgrades.crateQuality)
  const minLevel = Math.max(1, baseLevel)
  const maxLevel = Math.min(12, baseLevel + 4)
  eventBus.emit('rareCrate:opened', { x, y, minLevel, maxLevel })
  return  // <-- no frog spawn for rare box
}
// ... existing regular box frog spawn code follows ...
```

**8. In `create()` (or wherever other eventBus listeners are registered), add:**
```typescript
eventBus.on('rareCrate:claim', ({ level }) => {
  const state = useGameStore.getState()
  const loc = getLocationById(state.currentLocation)
  const { width, height } = this.scale
  const cx = width / 2
  const cy = height / 2
  const newFrog = this.spawnFrog(cx, cy, level)
  state.addFrogToLocation(loc.id, level)
  newFrog.container.setScale(0)
  this.tweens.add({
    targets: newFrog.container,
    scale: BASE_SCALE * 1.2,
    duration: 160,
    ease: 'Back.easeOut',
    onComplete: () => {
      this.tweens.add({
        targets: newFrog.container,
        scale: BASE_SCALE,
        duration: 100,
        ease: 'Power2.easeOut',
      })
    },
  })
})
```

**9. In `destroy()` or cleanup, add:**
```typescript
eventBus.off('rareCrate:claim')
```
(Check where other eventBus.off calls are — should be in the same place.)
