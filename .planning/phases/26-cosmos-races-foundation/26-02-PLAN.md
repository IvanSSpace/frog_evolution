    **Шаг A — Создать `client/src/game/data/habitablePlanets.ts`** — runtime API + deterministic seed-based selection algorithm (per CONTEXT D-DeterministicSeed).

    Содержимое:
    ```ts
    // Phase 26 Plan 26-02: habitable planets API.
    // 30/350 planets отмечены inhabitant'ами в planetMap.json (1 home + 2 colonies × 10 races).
    // Selection — ОДНОРАЗОВО на этом этапе (см. task 2.B script). После того как 
    // JSON committed — это runtime read-only API.
    
    import planetMap from './planetMap.json'
    import type { PlanetInhabitant } from '../../store/cosmic/types'
    import type { RaceId } from '../config/races'
    
    interface PlanetWithInhabitant {
      id: string
      name: string
      type: string
      x: number
      y: number
      size: number
      color: number
      inhabitant: PlanetInhabitant
    }
    
    /** Возвращает все habitable planets (30). Read-only — кешируется в module scope. */
    let cachedHabitable: PlanetWithInhabitant[] | null = null
    
    export function getHabitablePlanets(): PlanetWithInhabitant[] {
      if (cachedHabitable) return cachedHabitable
      const all = (planetMap as any).planets as Array<Record<string, any>>
      cachedHabitable = all.filter(p => p.inhabitant !== undefined) as PlanetWithInhabitant[]
      return cachedHabitable
    }
    
    /** Получить inhabitant для planet ID (undefined если uninhabited). */
    export function getPlanetInhabitant(planetId: string): PlanetInhabitant | undefined {
      const habitable = getHabitablePlanets()
      const found = habitable.find(p => p.id === planetId)
      return found?.inhabitant
    }
    
    /** Получить все planet'ы конкретной расы (length должен быть 3: 1 home + 2 colonies). */
    export function getPlanetsByRace(raceId: RaceId): PlanetWithInhabitant[] {
      return getHabitablePlanets().filter(p => p.inhabitant.raceId === raceId)
    }
    
    /** Список 30 IDs обитаемых планет (для quick lookups в renderer). */
    export const HABITABLE_PLANET_IDS: Set<string> = new Set(
      getHabitablePlanets().map(p => p.id)
    )
    ```

    **Шаг B — Selection algorithm + JSON write (one-time script logic).**

    Этот шаг performed ОДНОКРАТНО — нужно мутировать planetMap.json и закоммитить.     **Алгоритм (per CONTEXT specifics D-DeterministicSeed + D-AffinityPreference):**
    1. Seed = `19450718` (= planetMap.json meta.seed + 11 = derivative, документировать в habitablePlanets.ts).
    2. Mulberry32 PRNG (simple seedable, 6 lines кода — копируется в script).
    3. Race order — RACES array order (crystalloids, gasouls, ..., cometfolk).
    4. Element-to-race affinity mapping (см. Plan 26-01 race config: crystalloids→crystal, fireworms→fire, etc.).
    5. Для каждой race в порядке (deterministic):
       - Find все planets где `planet.type === race.affinity` AND `planet.id !== 'home'` AND planet ещё не assigned.
       - Если matching pool >= 3: shuffle PRNG'ом, take first 3. First = home, last 2 = colonies.
       - Если matching pool < 3: fallback — добрать из любых non-'home', non-assigned planets (PRNG random'ом). First из matching = home (если ≥1), иначе random = home.
       - Sort 2 colonies по distFromHome ASC (так colonies окажутся ближе к home planet'е расы — визуально логично).
    6. Output: для каждого выбранного planet ID — record `{id, role, raceId}`.
    7. Script записывает 30 inhabitant entries в planetMap.json (in-place patch — modify existing planets array, add `inhabitant: {raceId, role}` field; preserve все остальные fields untouched).
    8. Validation в script: assert 30 unique planet IDs, assert 10 home + 20 colonies, assert каждая race получила 3 planets, assert 'home' planet (id='home') НЕ assigned, assert все 320 untouched planets без inhabitant.

Expected: inhabited=30, homes=10, colonies=20.
