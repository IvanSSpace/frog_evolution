## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| LocalStorage → loadCosmicSlice | Untrusted JSON blob может содержать garbage в `firstContactsSeen` field |
| Server sync → cosmic blob | Сервер может прислать unknown raceId или non-boolean value |
| DEV helpers → window | Замусоривание global namespace, accidentally вызов в prod если gate не работает |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-26-01-01 | Tampering | `loadCosmicSlice()` firstContactsSeen parsing | mitigate | Defensive per-field validation: iterate over `defaults.firstContactsSeen` keys (known RaceId set), accept only `v === true`. Поломанный JSON → дефолт all-false. Pattern T-11-01. |
| T-26-01-02 | Tampering | Server-sync прислал unknown raceId | mitigate | `loadCosmicSlice` iterate'ит только known keys из defaults, unknown raceIds игнорируются |
| T-26-01-03 | Info Disclosure | DEV helpers in prod | mitigate | `import.meta.env.DEV` gate at function entry — installRaceDevHelpers early-returns в prod (vite tree-shakes DEV-blocks) |
| T-26-01-04 | DoS | Огромный firstContactsSeen blob | accept | Max size 10 booleans = ~150 bytes; не масштабируется к user input |

</threat_model>
